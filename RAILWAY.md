# Запуск бота на Railway (без терминала, всё мышкой)

Railway — это сайт, который сам берёт код из GitHub и держит бота включённым 24/7.
Никаких серверов и команд. Стоит примерно **5 $/мес**, нужна банковская карта.

Что уже готово: токен и твой Telegram ID (шаг 1 ты сделал). Поехали.

---

## Шаг 1. Завести код в главную ветку

Railway берёт код из главной ветки `main`. Я открыл для этого Pull Request —
тебе нужно один раз нажать кнопку слияния:

1. Открой ссылку на Pull Request (я её пришлю в чат).
2. Нажми зелёную кнопку **«Merge pull request»**, затем **«Confirm merge»**.

Готово — теперь весь код бота в главной ветке. Это делается один раз.

---

## Шаг 2. Регистрация на Railway

1. Зайди на **railway.com** → кнопка **Login**.
2. Выбери **Login with GitHub** и разреши доступ.

---

## Шаг 3. Создать проект из репозитория

1. На главной нажми **New Project**.
2. Выбери **Deploy from GitHub repo**.
3. Если репозитория нет в списке — нажми **Configure GitHub App** и дай Railway
   доступ к репозиторию **podslushano-bot**.
4. Выбери **podslushano-bot**. Railway начнёт собирать проект автоматически.

---

## Шаг 4. Вписать токен и настройки

1. Открой созданный сервис (плитка с названием) → вкладка **Variables**.
2. Нажми **New Variable** (или **Raw Editor**) и добавь три строки:

   ```
   BOT_TOKEN=сюда_токен_от_BotFather
   ADMIN_IDS=сюда_твой_id_от_userinfobot
   GUIDE_URL=https://www.podslushano.nl/contact-directory-netherlands/
   ```

3. Сохрани. Railway сам пересоберёт и перезапустит бота с этими настройками.

---

## Шаг 5. Чтобы база не обнулялась (важно)

У Railway файлы стираются при каждом обновлении. Чтобы заявки пользователей
не терялись, подключи постоянный диск:

1. В проекте нажми на сервис правой кнопкой (или **Settings** сервиса) →
   **Add Volume** / **Attach Volume**.
2. В поле **Mount path** впиши: **`/app/data`**
3. Сохрани.

(Специалисты не потеряются в любом случае — они каждый раз заливаются заново
из файла в коде. Диск нужен ради заявок и истории модерации.)

---

## Шаг 6. Проверить, что работает

1. В сервисе → вкладка **Deployments** → **View Logs**. Дождись строки
   **`Бот запущен`** (значит, всё хорошо; если есть ошибка — пришли мне текст).
2. Открой бота в Telegram → **/start** → кнопка поиска контактов →
   напиши *«нужен фотограф в Амстердаме»*. Бот вернёт реальных специалистов.
3. Заявки на модерацию будут приходить в личку тому, чей ID в `ADMIN_IDS`.

---

## Дальше само

- Любое изменение кода в `main` → Railway сам пересоберёт и перезапустит бота.
- Расходы видно в **Settings → Usage**. Выключить бота: поставить сервис на
  **Pause** или удалить (**Remove**).
