# Podslushano Bot 🇳🇱

Телеграм-бот — помощник по Нидерландам. Принимает от людей:

- 📰 анонимные истории / сплетни (для публикации в Instagram);
- ❓ вопросы в предложку;
- 🎬 видео от авторов контента;
- 📢 заявки на рекламу и сотрудничество;
- 🔍 помогает найти специалиста из гайда контактов (умный поиск по городу с
  подсказкой соседних городов/провинций).

Все заявки приходят администраторам **в личку** с кнопками
«✅ Одобрить / ❌ Отклонить».

```
│   └── db.py               # подключение к базе + первичное наполнение
│

│   ├── start.py            # /start, главное меню, кнопка «Отмена»
│   ├── submissions.py      # приём историй, вопросов, видео, рекламы
│   ├── contacts.py         # умный поиск специалиста (с уточняющими вопросами)
│   ├── moderation.py       # кнопки «Одобрить/Отклонить» у админа
│   └── chat.py             # живое общение: бот отвечает на ЛЮБОЕ сообщение
│
├── keyboards/
│   └── menus.py            # кнопки и меню
│
├── states/
│   └── forms.py            # «состояния» диалога (что бот ждёт от человека)
│
├── utils/
│   ├── geo.py              # города/провинции НЛ + категории специалистов

│
└── data/

```

## Запуск (пошагово)

### 1. Создать бота в Telegram
1. Напиши в Telegram боту **@BotFather** команду `/newbot`.
2. Придумай имя и username бота.
3. BotFather пришлёт **токен** — длинную строку. Скопируй её.

### 2. Узнать свой ID
Напиши боту **@userinfobot** — он пришлёт твой числовой ID.

### 3. Куда вставить токен — файл `.env`
Токен живёт в файле `.env` **на том компьютере/сервере, где запускается бот**.
В git этот файл не попадает (специально, чтобы секрет не утёк).

Пошагово:
1. В папке проекта найди файл `.env.example`.
2. Сделай его копию и назови её просто `.env` (с точкой в начале).
3. Открой `.env` любым текстовым редактором (Блокнот, TextEdit) и впиши:
```
BOT_TOKEN=сюда_токен_от_BotFather
ADMIN_IDS=твой_id,id_другого_админа
GUIDE_URL=https://www.podslushano.nl/contact-directory-netherlands/
```

⚠️ Токен — это ключ от бота. Никому его не пересылай и не публикуй.
Если токен куда-то «засветился» — напиши @BotFather команду `/revoke`,
он выдаст новый, а старый перестанет работать.

### 4. Установить библиотеки и запустить
```bash
pip install -r requirements.txt
python bot.py
```
Если всё хорошо — в консоли появится «Бот запущен». Напиши боту `/start`.

## Где брать данные специалистов

Сейчас в боте лежат **примеры-заглушки** (`data/specialists_seed.py`).
Их нужно заменить реальными контактами из гайда на сайте.
Категории должны совпадать со списком в `utils/geo.py` (`CATEGORIES`),
города — со списком там же (`CITY_TO_PROVINCE`).

> Чтобы пересоздать базу с новыми данными — удали файл `data/bot.db`
> и запусти бота снова.

## Хостинг (чтобы бот работал 24/7)

Пока бот запущен на твоём компьютере — он работает. Чтобы он работал
круглосуточно, его ставят на небольшой сервер (VPS, ~€4–6/мес).
Об этом — отдельным шагом, когда бот будет готов.
