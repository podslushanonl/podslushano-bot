  разное. Подача бесплатна (с модерацией), жильё — символически платно

  + ИИ-поиск событий по городу.
- 📚 **Полезное** — справочник о жизни в NL (BSN, DigiD, налоги, страховка…).
- 📩 **Разобрать письмо** — объяснение официальных писем по фото (ИИ-зрение).
- 🧮 **Калькулятор зарплаты** — расчёт netto.
- 💬 **Свободный чат** — отвечает на любые вопросы о жизни в NL (ИИ, с веб-поиском).
- 📰/❓/🎬/📢 **Предложка** — приём историй/сплетен, анонимных вопросов, видео,
  заявок на рекламу (через модерацию; одобренные вопросы — в канал).
- ➕ **Добавить себя в гайд** — платное размещение специалиста (Mollie).
- 📣 Поделиться ботом / рефералы · ✉️ поддержка/возвраты.

## Контент для соцсетей (для админов)
- 📝 **`/post`** — ИИ-пост в **Telegram-канал** по теме. Sonnet + веб-поиск,
  живой редакционный тон, без штампов и тире. Бот подбирает реальное фото со
  стока и публикует фото + подпись. В конце поста: призыв подписаться со
  ссылкой на канал + строка «ставьте 🔥». Перед публикацией — предпросмотр,
  кнопки **«✏️ Внести правки»** (правит этот же пост по указанию) и «переписать».
- 📸 **`/ig`** — **Instagram-карусель** (через Make). Бот сам:
  1. пишет контент (заголовок-хук + 5–8 слайдов, конкретика, веб-поиск);
  2. подбирает к каждому слайду реальное фото из **мультистока**
     (Pexels/Unsplash/Pixabay), а **ИИ-зрение выбирает** самый подходящий и
     эффектный кадр (без людей крупным планом, без повторов, умная обрезка);
  3. **рисует слайды 1080×1350** (Pillow): обложка, контент-слайды (заголовок
     + текст на фото с затемнением), и **фиксированный фирменный CTA**
     (косой срез фото→оранжевый, маскот-лосёнок со стикер-обводкой, шрифты
     Mont Heavy + Evolventa, постоянный фон, контакты IG/Telegram);
  4. ставит один хэштег (`#pnl_места` или `#pnl_нидерланды`);
  5. отдаёт готовые слайды по ссылке и шлёт JSON на **вебхук Make**, который
     публикует карусель в Instagram.
  - Кнопки: «🚀 Отправить в Make», «✏️ Внести правки», «🔁 Переписать»,
    «🔁 Отправить ещё раз» (если Make/Instagram не опубликовал).

## Контакт-гайд (каталог специалистов) — подробно
Проверенный каталог специалистов и услуг для русскоязычных в NL.

- **Категории:** ~45 категорий в 7 темах (здоровье, бьюти, дом, образование,
  вкус, дети, впечатления/услуги). Просмотр в 2 уровня: тема → категория.
- **Поиск:** по городу и категории; бот понимает синонимы/опечатки (ИИ),
  подсказывает соседние провинции, если в городе никого нет.
- **Карточка:** имя, категория, город/провинция, контакты, фото, отзывы и
  рейтинг. Выдача — каруселью (одна карточка, листание ◀️▶️) + «📤 Поделиться».
- **Премиум-выдача:** премиум-карточки показываются выше обычных.
- **Как попасть:** «➕ Добавить себя в гайд» — платное размещение (Mollie),
  на месяц или год, обычное или премиум, с фото. Продление по личной ссылке
  (claim/renew), бот шлёт напоминания об окончании.
- **Короткие ссылки:** карточку можно расшарить — бот отдаёт страницы
  `/c/<slug>` и `/s/<id>` (кнопка «📤 Поделиться»).
- **Сайт:** каталог на WordPress тянется из бота по JSON-фиду
  (`/api/specialists.json`).
- **Модерация/анти-скам:** ручная проверка добавлений; «услуги» только через
  платный гайд (на бесплатной доске их нет).

## Технологии
- **Python 3.11**, **aiogram 3**, **SQLAlchemy (async) + SQLite**.
- **aiohttp** веб-сервер: webhook оплаты Mollie, health-check, страницы
  `/privacy` `/terms` `/guide`, JSON-фиды для сайта, короткие ссылки
  `/c` `/s`, отдача готовых слайдов карусели `/ig-slide/<id>.jpg`.
- **ИИ:** Anthropic Claude — **Haiku 4.5** (чат, классификация запросов,
  выбор фото по зрению), **Sonnet 4.6** (разбор писем, посты `/post`,
  карусели `/ig`). Веб-поиск для свежих данных.
- **Картинки:** **Pillow** рисует посты-слайды (шрифты Oswald, Montserrat,
  Mont Heavy, Evolventa; маскот-лосёнок). Фото — стоки **Pexels / Unsplash /
  Pixabay** (ключи в env).
- **Instagram:** публикация через **вебхук Make** (`MAKE_WEBHOOK_URL`).
- **Оплаты:** Mollie (боевой). **E-mail:** Resend. **Хостинг:** Railway
  (автодеплой из ветки `main`). База — файл SQLite.

и `/terms`, RU/NL, с реквизитами). Бот видит только тех, кто сам написал/нажал
Start.
