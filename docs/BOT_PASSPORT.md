# 🪪 Паспорт бота «Подслушано в Нидерландах»

> Краткая справка о боте — чтобы дать контекст человеку или ИИ в любом чате.
> Обновлять при крупных изменениях.

## Кратко
Telegram-бот — помощник для **русскоязычных жителей Нидерландов** 🇳🇱.
Помогает с жизнью в NL, ищет специалистов, ведёт доску объявлений и афишу
мероприятий, отвечает на вопросы через ИИ. Часть функций — платные
(монетизация сообщества).

- **Сайт/сообщество:** https://www.podslushano.nl
- **Юрлицо:** Podslushano.nl · KVK 98882317 · BTW NL005359099B74 · Oss, NL
- **Поддержка:** podslushano.nl@gmail.com

## Что умеет (для пользователей)
- 🔍 **Найти специалиста** — умный поиск по контакт-гайду: по городу и
  категории, с подсказкой соседних провинций, отзывами и рейтингом, премиум-
  выдачей. Результаты — каруселью (одна карточка, листание ◀️▶️).
- 📋 **Объявления** — доска: жильё, вещи, отдам даром, работа, попутчики,
  разное. Подача бесплатна (с модерацией), жильё — символически платно (анти-
  скам). «Услуги» на доске НЕТ — они ведут в платный гайд. Платное «поднять».
- 📅 **Афиша** (бывш. «Чем заняться») — афиша мероприятий месяца (наши
  проверенные/платные события каруселью) + ИИ-поиск событий по городу.
- 📚 **Полезное** — справочник о жизни в NL (BSN, DigiD, налоги, страховка…).
- 📩 **Разобрать письмо** — объяснение официальных писем по фото (ИИ-зрение).
- 🧮 **Калькулятор зарплаты** — расчёт netto.
- 💬 **Свободный чат** — отвечает на любые вопросы о жизни в NL (ИИ).
- 📰/❓/🎬/📢 **Предложка** — приём историй/сплетен, анонимных вопросов, видео,
  заявок на рекламу (всё через модерацию; одобренные вопросы — в канал).
- ➕ **Добавить себя в гайд** — платное размещение специалиста (Mollie).
- 📅 **/afisha_add** — платное размещение мероприятия в афише.
- 📣 Поделиться ботом / рефералы · 🎨 стикеры · ✉️ поддержка/возвраты.

## Платные продукты (монетизация)
- Размещение специалиста в гайде (месяц/год, обычное и премиум) — Mollie.
- Размещение мероприятия в афише месяца — Mollie.
- Доска: размещение жилья (символическая плата) и «поднять объявление» — Mollie.
- Реклама / сотрудничество (заявки).
- Счета (factuur) на e-mail — через Resend.

## Для админов (основное)
- `/admin` — панель (база специалистов: добавить/найти/удалить, статистика,
  отзывы, рассылка, афиша в канал, ручное добавление в афишу).
- `/stats`, `/reviews`, `/broadcast` (рассылка всем), `/announce` (пост в канал).
- `/afishapost` (ИИ-афиша в канал), `/afisha_new` (афишу вручную),
  `/afisha_export ГГГГ-ММ` (сборка афиши месяца для Instagram).
- `/listing ID` (проверка объявления по жалобе), `/invoice`, `/grant`,
  `/premium`, `/setcity`, `/setcategory`, `/setcontact` и др.

## Технологии
- **Python 3.11**, **aiogram 3** (Telegram), **SQLAlchemy (async) + SQLite**.
- **aiohttp** веб-сервер: webhook оплаты Mollie, health-check, JSON-фид гайда
  для сайта.
- **ИИ:** Anthropic Claude — **Haiku 4.5** (чат, зарплата, события, афиша,
  классификация запросов), **Sonnet 4.6** (разбор писем по фото). Веб-поиск
  для свежих данных.
- **Оплаты:** Mollie (боевой режим). **E-mail:** Resend.
- **Хостинг:** Railway (автодеплой из ветки `main`). База — файл SQLite.

## Данные (таблицы)
`users` (пользователи для рассылок), `specialists` (гайд), `submissions`
(заявки предложки), `reviews` (отзывы), `events` (аналитика действий),
`event_listings` (афиша), `listings` (доска), `meta` (служебное).

## Приватность
Хранятся Telegram-id, username, имя, заявки, отзывы, действия. Политика
конфиденциальности и условия — на сайте (`/privacy`, `/terms`). Бот видит
только тех, кто сам написал/нажал Start.

## Интеграции
- **Telegram-канал** сообщества: автопубликация одобренных вопросов, анонсы,
  афиша.
- **Сайт (WordPress):** каталог специалистов может тянуться из бота по JSON-фиду.

## Принципы/тон
Тёплый, дружелюбный, по-русски, по делу. Помощь с жизнью в NL — бесплатно;
размещение бизнесов/специалистов/мероприятий — платно. Анти-скам и модерация
там, где есть риск (жильё, доска, предложка).
