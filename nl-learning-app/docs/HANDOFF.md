## 1. Что я понял о проекте (кратко)
Делаем **бесплатное приложение для изучения нидерландского на русском** для русскоязычных
в Нидерландах, цель — довести до речи и до уровня инбургеринга (A2→B1), монетизация — PRO 9,99 €/мес.

ядро — **SRS (повторения)**, активная продукция (говорение/письмо) и **AI-репетитор на реальной LLM**.
Структура курса A1 берётся из **TaalCompleet A1** (8 тем), грамматика — из книги Миронова.

## 4. Какие команды я запускал (как проверять)
Прототип — один HTML-файл; проверяю так:
1. **Синтаксис JS:** вырезать содержимое `<script>` из HTML и прогнать
   `node --check` (см. пример ниже).
2. **Рендер/поведение:** запускать в **headless Chromium** через Playwright
   (`/opt/node22/lib/node_modules/playwright`), проверять `#screen`, отсутствие ошибок консоли,
   переходы, счётчики.
3. Правки контента вносил **скриптами на Python** (точечные строковые замены в HTML),
   чтобы не переписывать весь файл и не ломать сборку.

Пример проверки синтаксиса:
```bash
python3 - <<'PY'
import re; h=open('appprototype.html',encoding='utf-8').read()
open('_check.js','w').write(re.search(r'<script>(.*)</script>', h, re.S).group(1))
PY
node --check _check.js && echo OK
```

## 5. Следующий конкретный шаг (рекомендация)
> Приоритет — уйти от «прототип-монолита» и от клона Duolingo, при этом продолжать наполнять A1.

**Шаг 1 (архитектура):** вынести контент из HTML в **данные (JSON)** по схеме
`theme → lesson → part → step` и `word {nl, ru, theme, unit}`. Это разблокирует масштаб.
**Шаг 2 (UI):** переделать карту на **единую непрерывную ленту тем** (не вкладки сверху),
убрать лишние плашки, задать **собственный визуальный стиль** (не Duolingo).
**Шаг 3 (контент):** запустить **AI-конвейер** — генерировать переводы + 5–15 предложений/заданий
на каждое слово из TaalCompleet A1, наполнять темы 3–8, идти к 1500 словам.
**Шаг 4 (система):** реальная персистентность прогресса/SRS (даты SM-2), затем реальная LLM для AI и оплата PRO.

## 6. Где могут быть проблемы
- **Монолит:** весь контент в одном HTML — тяжело редактировать и версионировать. Вынести данные в первую очередь.
- **Нет персистентности:** прогресс/SRS/XP в памяти, сбрасываются при перезагрузке.
- **Моки:** лига/лента/друзья/оплата/AI — без бэкенда/LLM. Не считать их готовыми.
- **CSP артефакта:** прототип не грузит внешние ресурсы (шрифты/картинки/CDN) — реальные фото и
  арт персонажа в этом HTML не вставить; в проде нужны бандл-ассеты.
- **Аудио:** TTS/STT через браузер; голос `nl-NL` есть не на всех устройствах — нужен фолбэк.
- **Не путать с ботом:** репозиторий `podslushano-bot` — другой продукт; правки языка — только в `nl-learning-app/`.
