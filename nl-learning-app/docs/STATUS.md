## ✅ Что уже сделано и работает (в прототипе)
- **Онбординг-воронка:** приветствие → цели (мультивыбор) → уровень (сам/тест) →
  формат (5/10/15/20 мин) → экран PRO (9,99 €, 14 дней бесплатно).
- **Сплеш-анимация** запуска.
- **Тест уровня:** 12 вопросов A1→B1, варианты перемешиваются, итог → уровень A1/A2/B1.
- **Карта в стиле Duolingo:** 8 тем-модулей, извилистая тропа, узлы
  (⭐ урок / 🏋️ практика→SRS / 🎁 сундук→алмазы / 🏆 трофей), цветной баннер темы,
  подсветка текущего узла со «СТАРТ».
- **Уроки → части:** каждая часть проходится отдельно, по порядку.
- **11 типов заданий** работают: explain, word, quiz, pick, listen, gapType, translate, tf, build, match, speak.
- **Движок урока:** прогресс-бар, «жизни» (сердца), ошибки повторяются в конце части,
  редкая похвала на нидерландском (Goedzo!/Uitstekend!…), конфетти на финише.
- **SRS (интервальные повторения):** «Умное повторение», сила памяти (6 уровней),
  даты повтора, экран «Память», демо-кнопка «следующий день».
- **Банк слов:** 170 слов; из них 160 — словарь **TaalCompleet A1 по 8 темам** (с переводами и озвучкой).
- **Экран «Словарь A1 по темам».**
- **Геймификация (визуально):** XP, стрик 🔥, алмазы 💎, сундуки, лига, лента, друзья, достижения.
- **Инбургеринг-трекер** в профиле: Lezen/Luisteren/Spreken/Schrijven/KNM (честный расчёт от прогресса и точности).
- **Озвучка (TTS)** через SpeechSynthesis `nl-NL`; **распознавание речи (STT)** через SpeechRecognition в говорении.
- **Адаптив:** на телефоне приложение разворачивается на весь экран.
- Проверено в **headless Chromium** (Playwright): рендер, переходы, отсутствие ошибок консоли.

## 🟡 Что частично работает / это мок (нужен бэкенд или доработка)
- **AI-репетитор** — экран-чат с **заготовленными ответами** (canned). Нужна реальная LLM.
- **Лига / Лента / Друзья** — визуальные макеты, **без сервера** (данные захардкожены).
- **Оплата PRO** — только экран, платёжной интеграции нет.
- **Проверка произношения** — базовая (распознал слова / нет), без реальной оценки фонетики.
- **SRS** — работает в рамках сессии; **нет персистентности** между запусками и реальных дат
  (используется «виртуальный день» + демо-кнопка).
- **Прогресс, XP, SRS, gems** — **всё в памяти**, сбрасывается при перезагрузке страницы.

## 🗂️ Банк слов (данные)
- **`data/taalcompleet_a1_woordenlijst.json`** — **официальный словарь TaalCompleet A1** (674 слова,
  8 тем, с артиклями de/het, мн. числом, под-юнитами 1.1…). Распарсен из официального PDF;
  переводы: 567 подтянуты из базы 5000 + 108 добавлены вручную. **Это канонический A1-словарь.**
- Встроен в приложение как `WL` → экран **«Словарь A1 · TaalCompleet»** (8 официальных тем) + сидирует SRS.
- **`data/source/`** — исходные материалы как есть: `combined_all_pdf_text_raw.{md,json}`
  (сырой текст A1-словаря + рабочих листов A2), `README_FOR_CLAUDE.md`, `MANIFEST.json`.
  ⚠️ A2-рабочие листы пока не распарсены (нужен отдельный парсер).

- **`data/dutch_a1_a2_5000.json`** — полная база **5000 единиц A1–A2** (слова, формы, фразы,
  предложения) с русскими переводами. ⚠️ Переводы слов чистые, но у фраз/предложений/форм —
  **черновые/шаблонные** («Я ищу: мужчина.») — нужна AI-вычитка перед показом.
- В прототип встроен **чистый срез — 413 слов** (существительные+артикль, глаголы-инфинитивы,
  прилагательные, цвета), размеченный по 8 темам → структура `BANK` в `app.html`.
- Экран **«Словарь A1–A2»** показывает все 413 слов; они же **сидируют SRS** (`srsLoadBank`).
