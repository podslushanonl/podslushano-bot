# Компоненты для будущего React Native / Expo приложения

Валюта приложения — **Stroopwafels 🧇**.
- `StroopwafelToken.tsx` — SVG-монета (react-native-svg): `npx expo install react-native-svg`
- `StroopwafelBalance.tsx` — чип баланса с кнопкой «+»
Цвета: caramel #D56A09 · waffle #F3A535 · dark brown #3A1D0A · highlight #FFE3A3.
В HTML-прототипе используется адаптированная inline-SVG версия (функция `stroop(size)` в app.html).
